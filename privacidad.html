1. RESPONSABLE DEL TRATAMIENTO

El responsable del tratamiento de los datos personales recogidos a través de este sitio web es el titular de cambiademarcha.es, contactable en [email protected].

2. DATOS QUE RECOGEMOS

Este sitio web no requiere registro ni recoge datos personales de forma directa. No obstante, pueden recogerse de forma automática los siguientes datos de carácter técnico:

  • Dirección IP del visitante
  • Tipo de navegador y dispositivo
  • Páginas visitadas y tiempo de permanencia
  • País de origen de la visita

Estos datos son recogidos de forma anónima y agregada con fines estadísticos y de mejora del servicio.

3. FINALIDAD DEL TRATAMIENTO

Los datos técnicos recogidos se utilizan exclusivamente para:

  • Analizar el uso del sitio web y mejorar su funcionamiento
  • Detectar y corregir errores técnicos
  • Gestionar la publicidad mostrada a través de Google AdSense

4. GOOGLE ADSENSE Y PUBLICIDAD

Este sitio web utiliza Google AdSense para mostrar publicidad. Google, como proveedor externo, utiliza cookies para mostrar anuncios basados en las visitas anteriores del usuario a este sitio web y a otros sitios de Internet.

Puedes desactivar el uso de cookies por parte de Google visitando la página de configuración de anuncios de Google. También puedes desactivar las cookies de terceros visitando aboutads.info.

5. BASE JURÍDICA

El tratamiento de datos se realiza al amparo del interés legítimo del titular del sitio en el análisis del tráfico web y la mejora del servicio, conforme al artículo 6.1.f del Reglamento General de Protección de Datos (RGPD).

6. CONSERVACIÓN DE DATOS

Los datos técnicos recogidos de forma anónima se conservan durante el tiempo necesario para cumplir con la finalidad para la que fueron recabados, sin exceder los plazos legalmente establecidos.

  • Acceder a sus datos personales

  • Solicitar la supresión de sus datos
  • Oponerse al tratamiento
  • Solicitar la limitación del tratamiento
  • Presentar una reclamación ante la Agencia Española de Protección de Datos (AEPD)
